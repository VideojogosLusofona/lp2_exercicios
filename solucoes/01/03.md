## Problema

3 - Indica o que é mostrado no ecrã pelo seguinte código C#. Justifica a tua resposta.

```cs
byte b = byte.MaxValue - 1;

for (int i = 0; i < 5; i++)
{
    Console.WriteLine($"Iteração {i}: {(byte) (++b - 1)}");
}
```

## Soluções

### Solução 1

```
Iteração 0: 254
Iteração 1: 255
Iteração 2: 0
Iteração 3: 1
Iteração 4: 2
```
